# Jogo-do-que-mesmo? 🧠 🤔

> Repositório do melhor jogo da memória do Brasil! 

Uma aplicação simples, divertida e nostálgica para testar sua capacidade de memorização (ou descobrir se você precisa começar a anotar tudo).

---

## 📋 Sobre o Projeto

O **Jogo-do-que-mesmo?** é um jogo da memória interativo desenvolvido em Python. O objetivo é desafiar o jogador a encontrar os pares de cartas no menor tempo ou número de jogadas possível. Ideal para treinar o cérebro ou apenas passar o tempo!

### ✨ Funcionalidades
*   [x] Interface dinâmica (Gráfica).
*   [x] Contagem de tentativas/erros.
*   [x] Pontuação e ranking local de jogadores.
*   [x] Sistema de reinicialização rápida.
*   [x] Até 3 dificuldades (a difícil é impossível!!!)

## 🔧 Instalação e Execução

Você pode rodar o projeto na sua máquina de duas formas:

### Opção 1: Via Download Direto (Para não-desenvolvedores)
1. Acesse o repositório público: [Jogo da Memória](https://github.com/IcaroCaminoto/Jogo-da-Memoria/tree/main).
2. Clique no botão verde **"<> Code"** e selecione a opção **"Download ZIP"**.
3. Extraia a pasta `Jogo-da-Memoria-main` do arquivo compactado.
4. Abra a pasta extraída e execute o arquivo `main.py`.

### Opção 2: Via Terminal (Para desenvolvedores)
Se você já usa o Git, basta clonar o repositório:

```bash
# Clonar o repositório
git clone [https://github.com/IcaroCaminoto/Jogo-da-Memoria.git](https://github.com/IcaroCaminoto/Jogo-da-Memoria.git)

# Entrar na pasta do projeto
cd Jogo-da-Memoria

# Executar o jogo
python main.py
